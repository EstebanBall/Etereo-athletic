Etereo Athletic

Proyecto de e-commerce de suplementos desarrollado tanto para el aprendizaje de arquitecturas escalables y automatización, como para la entrega de pagina funcional.

Tecnologías utilizadas:
- Frontend: Next.js 15 (App Router), React, Tailwind CSS.
- Lenguaje: TypeScript.
- Arquitectura: Feature-First (Modular).

Estructura del Proyecto
El proyecto utiliza una estructura de carpetas basada en funcionalidades (Features), lo que permite una fácil migración a microservicios en el futuro.

Progreso actual
Semana 1:
Lunes: Configuración inicial, limpieza de boilerplate, creación de componentes UI base y maquetación de la Home Page con grid de productos estático.
